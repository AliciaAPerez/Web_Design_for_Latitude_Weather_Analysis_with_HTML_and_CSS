Summary: Latitude vs. X

The purpose of this dataset is to compare the temperature, humidity, cloudiness, and wind speed against the latitude of the city involved. With the latitude, we can see how the data changes as we move closer and farther away from the equator.

The dataset used was a randomized list of over 500 cities with the weather information being pulled from the OpenWeatherMap API. Matplotlib was used to anazlye the date into graphs. All of the data can be viewed on the data page. Please view the comparison page and each visualization page for more information on each weather factor recorded.
